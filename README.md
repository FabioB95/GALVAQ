## Description
This repository contains the code and LaTeX source for the paper "GALVA-Q: A Hybrid Quantum--Machine Learning Framework for Electroplating Slot Optimisation on Industrial Cylindrical Racks" by Fabio Buffoli.

**Note**: This code is released for research reproducibility. Commercial use requires prior permission. See LICENSE for details.

## Repository Contents
- `code/` — Python implementation (QUBO builder, QAOA on Quafu, XGBoost prior, greedy placement, Faraday model)
- `galvaq_paper.tex` — Full Overleaf-compatible LaTeX source
- `galvaq_full_project.zip` — Complete archive of everything (recommended download)

## How to Run
1. `pip install -r code/requirements.txt`
2. Run `python code/main_pipeline.py`
